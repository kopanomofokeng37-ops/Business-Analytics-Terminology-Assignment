# Business-Analytics-Terminology-Assignment
This assignment is based on define, explaining and providing example of different business analytic terms.

****📌 Project Overview****

This repository contains my completed **Business Analytics Terminology Research Assignment**, focused on understanding the core vocabulary, KPIs, and analytical concepts used in modern business environments.

The project explores how organizations use data to measure performance, make informed decisions, improve customer acquisition, optimize marketing campaigns, and forecast future outcomes.

This assignment was completed as part of a foundational **Business Analytics learning programme*


**🎯Objectives**

- Build a strong understanding of essential business analytics terminology
   
- Distinguish between commonly confused concepts (e.g. KPI vs Metric, BI vs Analytics)
  
- Apply analytical terms to real-world business examples
  
- Understand customer, sales, and marketing performance metrics
  
- Strengthen business communication and data literacy skills
